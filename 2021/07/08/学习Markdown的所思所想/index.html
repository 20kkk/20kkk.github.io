# 学习Markdown的所思所想

  • 一、我对Markdown的认识

使用Markdown易于编写以文本格式的文章，以人们喜闻乐见的方式表达出来，具有条理性；对于Markdown一些快捷键比如创建标题级数只使用即可，还有加粗，斜体，代码块等等，对于写文章而言，比以前使用的Word有着事半功倍的效果。我认为Markdown语言使用思路清晰，掌握其基本的写作思路便可写出一篇条理清晰的文章

  • 二、Markdown的使用方法

字体：* 标示斜体，* **标示粗体,~~ 删除文本~~。
段落：# 标示一级标题、## 标示二级标题、如此增加到六级标题，> 标示段落引用，- 标示无序列表等等；

表格:

姓名 年龄 成绩
张三 19 111
李四 20 222
我 19 333

图片:！[图片名称]（图片地址） ————-这里的图片地址从百度图床中获取

连接：[链接注释]（链接地址）

  • 三、对使用Markdown中的不足

对于Markdown的使用并非一朝一夕就可熟练上手，想要学好并熟练应用Markdown语言，应该多写文章，多应用，他没有复杂的逻辑关系，它是一种格式的表达形式，应用这中语言可以让文章有理，有条结构框架一目了然。
